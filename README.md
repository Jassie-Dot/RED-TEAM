# VIGIL-AI (Employer Mode)

Working MVP for mock resume validation and skill testing.

## Run

```bash
npm run dev
```

Open `http://localhost:3000`.

## MVP Flow

1. Landing page with Employer Mode and disabled Student Mode
2. Mock resume upload
3. Mock skill extraction for John Doe
4. Skill-based test generation
5. Answer evaluation through API route
6. Employer report with claimed vs actual comparison
